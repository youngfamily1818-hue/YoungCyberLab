# In the Frequency Clouds

## Objective

Analyze an audio file to discover hidden information and recover the challenge flag.

## Tools Used

* Sonic Visualiser
* Spectrogram Analysis
* Linux
* Basic Steganography Concepts

## Process

1. Downloaded the provided audio file.
2. Opened the file in Sonic Visualiser.
3. Switched to spectrogram view.
4. Examined frequency patterns for hidden visual information.
5. Identified characters embedded within the spectrogram.
6. Extracted the hidden message and recovered the challenge flag.

## Skills Demonstrated

* Audio Steganography
* Spectrogram Analysis
* Linux Tool Usage
* Digital Forensics
* Problem Solving

## Lessons Learned

This challenge demonstrated how information can be hidden within audio files and highlighted the importance of spectrogram analysis techniques during forensic investigations and Capture The Flag exercises.
