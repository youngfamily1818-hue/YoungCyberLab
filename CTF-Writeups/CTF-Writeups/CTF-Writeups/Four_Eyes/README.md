# Four Eyes Steganography Challenge

## Objective

Recover hidden information concealed inside an image file.

## Tools Used

* Steghide
* PowerShell
* Linux
* Basic Steganography Techniques

## Process

1. Downloaded the challenge image.
2. Installed Steghide.
3. Attempted extraction using the provided image file.
4. Investigated possible passwords and metadata.
5. Extracted hidden content from the image.
6. Decoded the recovered information to obtain the challenge solution.

## Skills Demonstrated

* File Analysis
* Steganography
* Linux Command Line
* PowerShell
* Problem Solving

## Lessons Learned

This exercise demonstrated how files can contain hidden information and emphasized the importance of steganography analysis techniques used in digital forensics and cybersecurity investigations.
