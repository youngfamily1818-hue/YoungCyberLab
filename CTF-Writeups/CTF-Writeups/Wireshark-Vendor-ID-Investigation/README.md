# Unregistered Laptop Investigation

## Objective

Identify the vendor of an unauthorized laptop discovered on a restricted corporate network.

## Tools Used

* Wireshark
* Packet Capture Analysis
* MAC Address Investigation

## Process

1. Opened the provided packet capture in Wireshark.
2. Examined network traffic associated with the suspicious device.
3. Located the MAC address of the unknown system.
4. Identified the Organizationally Unique Identifier (OUI).
5. Used the OUI information to determine the vendor of the device.
6. Confirmed the laptop manufacturer for incident reporting purposes.

## Skills Demonstrated

* Wireshark Analysis
* Packet Inspection
* MAC Address Investigation
* Network Forensics
* Incident Response

## Lessons Learned

This challenge reinforced the importance of packet analysis and MAC address identification during network investigations. Even limited information can provide valuable intelligence for identifying unauthorized devices.
